                              Navigation Support

Our Navigation App is a local navigation application for iOS with user-friendly interface and powerful function. You can use this app when you drive a car, ride a bike or walk to your destination. This app provides local search and offline navigation service which is especially for you when you travel or live in this country.The navigation route is optimized by shortest path algorithm which can reduce your travel time.

Key Features:

1.Offline map with high quality.

2.Offline navigation guide you to destination with direction,distance,velocity during navigation.

3.Professional navigation engine makes you a good navigation experience and save much time.

4.Smart route in big cities of this country.

5.Powerful voice navigation, provide most clear route guide for you. View the route on the table list or show route animation on the map

6.provide offline,nationwide,comprehensive,accurate,search service.

7.Friendly user-interface, perfectly compatible with iOS 7 style.

8.The map sources from OpenStreetMap, we will update the map data regularly.

9.Built-in electronic compass can indicate direction.

10.Adopt some advanced technology such as vector map,shortest path algorithm. Continued use of GPS running in the background can dramatically decrease battery life.
